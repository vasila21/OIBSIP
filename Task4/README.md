This Online Examination System is a Java Swing based desktop application developed to conduct simple online tests through an interactive graphical user interface. The application provides a secure login system where users must enter valid credentials before accessing the examination portal. After successful login, users can attend multiple-choice questions, select answers using radio buttons, navigate through questions, and submit the exam digitally.

The system includes a timer functionality that automatically submits the exam when the allotted time is completed, making the examination process more realistic and efficient. It also calculates the final score instantly based on the user’s selected answers and displays the result after submission. The application supports profile updating, allowing users to change their username and password directly from the interface. Additionally, the logout feature provides secure exit confirmation for better user control.

The project is developed using Java, Java Swing, AWT Event Handling, Timer class, and Object-Oriented Programming concepts. The graphical interface is designed to be beginner-friendly, responsive, and easy to use for students learning Java GUI application development.
