This Number Guessing Game is a Java Swing based desktop application developed to provide an interactive and fun number prediction experience for users through a graphical user interface. The game generates a random number between 1 and 100, and the player must guess the correct number within a limited number of attempts. After every guess, the system provides hints such as “Too High” or “Too Low” to help the player reach the correct answer.

The application includes a scoring system where players earn points based on the number of attempts used to guess the correct number. Fewer attempts result in higher scores, making the game more challenging and engaging. The game also supports multiple rounds, allowing users to continue playing without restarting the application. A separate “New Game” option resets the current game instantly, while the “Exit” option safely closes the application with a confirmation dialog.

The project is developed using Java, Java Swing, AWT Event Handling, Random class, and Object-Oriented Programming concepts. The graphical interface is designed to be simple, responsive, and beginner-friendly for students learning Java GUI application development.

Score Calculation System,
