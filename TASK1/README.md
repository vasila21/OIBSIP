This Online Reservation System is a Java Swing based desktop application developed to simplify the process of train ticket booking and cancellation through an interactive graphical user interface. The system provides a secure login module where users must enter valid credentials to access the reservation portal. After login, users can fill in passenger details such as passenger name, train number, train name, class type, journey date, source station, and destination station to reserve tickets successfully.

The application automatically generates a unique PNR number for every reservation, making ticket identification simple and organized. It also includes proper input validation to ensure that users cannot submit empty fields or invalid information. A separate cancellation module allows users to search tickets using the generated PNR number, view complete reservation details, and cancel tickets easily.

The project is developed using Java, Java Swing, AWT event handling, ArrayList for temporary data storage, and Object-Oriented Programming concepts. The graphical interface is designed to be user-friendly and easy to understand for beginners learning Java GUI development.
